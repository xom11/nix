# Karabiner-Elements

> *Config keyboard in macos*

---
### Tab behavior
- Remap `Tab` to `Cmd + Ctrl + Opt + Shift` when holding
- `Tab + <number>` to paste from clipboard history
- `Tab + H/J/K/L` to move left/down/up/right
- `Tab + Y/U/I/O` to remap to Home/PgDown/PgUp/End
- `Tab + ~` to area selection snapshot 
### Caplock behavior
- Remap `Caps Lock` to `Escape` when tapping
- Remap `Caps Lock` to `Cmd + Ctrl + Opt` when holding
- `Cap + <number>` to copy to clipboard
- `Cap + <char>` to run or focus app (using raycast)
