# Shortcuts

> Config keyboard in macOS/Linux

---

## System

### Tap vs Hold Modifier Keys

- Remap `Tab` to `hypr` when holding
- Remap `Caps Lock` to `Escape` when tapping
- Remap `Caps Lock` to `Cmd + Ctrl + Opt` when holding
- Remap `Right Shift` to `paste` when tapping
- Remap `Left Ctrl` to `copy` when tapping

### App Launcher

- `Cap + <char>` to run or focus app

### Clipboard Manager

| Shortcut | Action |
|---|---|
| `Cap + <number>` | Copy to clipboard `<number>` (1-5) |
| `Tab + <number>` | Paste from clipboard `<number>` |
| `Alt + V` | Open clipboard history |
| `Tab + ~` | Area selection snapshot to clipboard |

### Arrow Keys and Navigation

| Shortcut | Action |
|---|---|
| `Tab + H/J/K/L` | Move left/down/up/right |
| `Tab + Y/U/I/O` | Home/PgDown/PgUp/End |

### Audio and Brightness

| Shortcut | Action |
|---|---|
| `Cap + plus/minus` | Volume up/down |
| `Cap + 0` | Mute/unmute |
| `Tab + plus/minus` | Brightness up/down |
| `Tab + 0` | Mic mute/unmute |

### Power and Lock

| Shortcut | Action |
|---|---|
| `Meta + Alt + L` | Lock screen |
| `Meta + Alt + Shift + L` | Lock out |
| `Meta + Alt + Shift + S` | Shutdown |
| `Meta + Alt + Shift + H` | Hibernate |
| `Meta + Alt + Shift + R` | Reboot |

### Common Shortcuts (Space as modifier)

| Shortcut | Action |
|---|---|
| `Space + A` | Select all |
| `Space + C` | Copy |
| `Space + X` | Cut |
| `Space + V` | Paste |
| `Space + Z` | Undo |
| `Space + S` | Save |
| `Space + F` | Find |
| `Space + W` | Close window |
| `Space + Q` | Quit app |
| `Space + R` | Refresh/reload |

### Screenshot

- `Tab + ~` to take screenshot

### Hide All Windows

- `Meta + Alt + H` to hide all windows
